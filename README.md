# 学习激励系统

一个基于 Vue 3 + TypeScript 开发的学习激励应用，通过任务管理、积分系统、成就系统和抽奖功能，帮助用户培养良好的学习习惯。

## 功能特性

### 📚 核心功能
- **任务管理**：支持语文、数学、英语三科的日常学习任务管理
- **积分系统**：完成任务获得积分，可用于兑换物品和抽奖
- **成就系统**：多种成就类型，激励用户持续学习
- **抽奖功能**：消耗积分参与抽奖，获得不同稀有度的物品
- **背包系统**：管理获得的物品，支持使用物品获得特殊效果
- **排行榜**：展示用户积分排名，激发竞争意识
- **数据统计**：查看学习数据和进度

### 🏆 成就类型
- **学习新手**：完成第一个学习任务
- **学习达人**：完成10个学习任务
- **学习大师**：完成50个学习任务
- **积分系列**：累计获得指定积分
- **连续学习**：连续学习3天、7天、14天、30天
- **完美一周**：一周内完成所有学习任务
- **多才多艺**：完成所有科目的学习任务
- **抽奖幸运儿**：抽中5次稀有物品

### 🎰 抽奖系统
- **物品稀有度**：普通（70%）、稀有（20%）、史诗（8%）、传说（2%）
- **特殊效果**：稀有物品可提供额外积分或学习加速
- **抽奖消耗**：每次抽奖消耗10积分

## 技术栈

- **前端框架**：Vue 3
- **开发语言**：TypeScript
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **构建工具**：Vite
- **样式**：原生 CSS

## 使用说明

### 登录
- 默认用户名：`xumingxi`
- 默认密码：`20160104`

### 任务管理
1. 在任务页面查看当日学习任务
2. 点击任务卡片可编辑任务描述
3. 根据完成情况选择完成度（低/中/高）
4. 完成任务后获得相应积分

### 积分兑换
1. 在积分页面查看可兑换物品
2. 点击兑换按钮消耗积分获得物品
3. 兑换的物品将存入背包

### 抽奖功能
1. 在积分页面点击抽奖按钮
2. 消耗10积分参与抽奖
3. 获得随机物品，存入背包

### 成就系统
1. 在成就页面查看所有成就
2. 查看已解锁和未解锁的成就
3. 查看成就进度和解锁条件

### 背包管理
1. 在个人资料页面查看背包物品
2. 点击物品可查看详细信息
3. 点击使用按钮使用物品，获得特殊效果

## 核心功能说明

### 任务完成度与积分对应关系
- **低完成度**：1积分
- **中完成度**：2积分
- **高完成度**：3积分

### 连续学习计算
- 连续每天完成至少一个任务，连续天数递增
- 中断一天，连续天数重置为0

### 成就解锁奖励
- 解锁成就时获得10积分奖励
- 成就解锁后显示解锁时间

### 物品稀有度效果
- **普通物品**：基础奖励
- **稀有物品**：使用后获得5点积分或额外任务完成度
- **史诗物品**：使用后获得10点积分或双倍积分效果
- **传说物品**：使用后获得20点积分和额外抽奖机会，或解锁所有成就进度+1
